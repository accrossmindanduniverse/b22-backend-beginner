# **Coffee Shop App API**

**Equipments**
1. Node Js
2. Express Js
3. API testing tools (Postman, etc)

**Roles**
1. user
2. admin


**CRUD**
1. User
2. Product
4. Chat
3. Category
4. Authentication
5. Transaction


**How to run the app**
1. Open app's directory in Terminal or Command Prompt
2. Type npm install
3. Create .env file to set up the app
4. run the app using yarn or npm

**.env set up**
```
NODE_ENV=development

APP_URL=
APP_KEY=

APP_UPLOAD_ROUTE=
APP_UPLOAD_PATH=
APP_TRANSACTION_PREFIX=

MYSQL_HOST=localhost
MYSQL_USER=root
MYSQL_PASSWORD=
MYSQL_DATABASE=coffee_shop
```
